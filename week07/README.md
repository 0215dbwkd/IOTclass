블로킹(Blocking) 
  특정 기능 수행에 로직을 묶어 놓아서 다른 일을 못하게 되면 프로그램이 블로킹되었다고 한다
반대로 여러 기능을 수행하는 것은 논블로킹(non-blocking) 방식이다

인터럽트(Interrupt) 
제일 급한 신호부터 처리한다  

cpp 파일은 실행 파일이다. 정의를 포함하는 파일
h 주로 선언을 포함하는 파일 

같은 이름에 '매개 변수형'이나 '개수'를 다르게 갖는 것을 "함수 오버로딩(function overloading)이라고 한다.
